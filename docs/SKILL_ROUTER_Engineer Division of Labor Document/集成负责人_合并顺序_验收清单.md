## 4. 高风险冲突文件

| 文件 | 冲突原因 | 处理方式 |
|---|---|---|
| `config.yaml` | 多人可能加配置 | 以工程师 A 的结构为准 |
| `.env.example` | 环境变量 | 保留 `ES_INDEX` 和 `SKILL_ROUTER_ES_INDEX` |
| `skills/registry.json` | B/C/F 都可能更新 | 保留最新 schema 和状态字段 |
| `backend/packages/harness/deerflow/agents/lead_agent/agent.py` | Middleware 接入 | SkillRouter 必须在 TodoMiddleware 前 |
| `backend/packages/harness/deerflow/skills/loader.py` | Loader 过滤 | 保持兼容原行为 |
| `Makefile` | 多人加命令 | 合并所有命令 |
| `skills/**/router_card.json` | B/F 都可能生成 | 保留人工校准版本 |

# Road Traffic Schema (Scaffold)

This folder stores schema notes and field mappings for road traffic datasets.

Suggested canonical fields (adjust later):

- `timestamp`
- `road_segment`
- `intersection_id`
- `direction`
- `vehicle_count`
- `avg_speed`
- `occupancy`
- `travel_time`
- `incident_flag`
- `source_file`

Use `field_mapping.yaml` to map source columns into these canonical names.

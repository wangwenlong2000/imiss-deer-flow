# common_neighbor 在电话网络任务里的落地说明

## 图论原义

common_neighbor 指的是：

> 给定两个节点，找出同时和它们都相连的节点集合。

## 在电话网络里的具体对应

### 号码-号码图
如果把号码看作节点，那么两个号码的 common_neighbor 就是：

- 两个号码都联系过的共同对端

### 号码-设备关系图
如果把号码和设备关系抽成二部图，那么两个号码的“共享设备”也可以视作一种特殊的共同邻居分析：

- 两个号码都连到同一个设备节点

## 为什么它重要

因为在风控排查里，两个号码即使没有直接通话：

- 只要它们有大量共同对端
- 或共享同一个设备

就说明它们不一定是孤立的，可能处于同一个局部关系圈、设备池或接触目标集里。

这正是 overlap-analysis skill 的核心任务。

# 异常检测算法说明

## Z-score

适合检测全局极端值：

```text
z = (x - mean) / std
```

默认阈值为 `3.0`。

## MAD

适合抗离群点的稳健异常检测：

```text
robust_z = 0.6745 * (x - median) / MAD
```

默认阈值为 `3.5`。

## IQR

适合检测箱线图意义上的上下界异常：

```text
lower = Q1 - k * IQR
upper = Q3 + k * IQR
```

默认 `k = 1.5`。

## ESD

本 Skill 使用简化版 ESD 思路：

1. 在每个分组内计算当前最大 z-score。
2. 若超过阈值，则标记该点并从候选集中移除。
3. 重复直到达到最大异常比例或没有超过阈值的点。

该实现更适合工程筛查，不等价于严格统计检验中的完整临界值表。

## Rolling-Z

按时间排序后，使用最近 `window` 条历史记录计算均值和标准差，检测当前点是否偏离历史窗口。

适合：

- 签到量突增
- 区域活跃度突然下降
- 用户出行距离异常增大

## LOF

局部离群因子基于多个数值指标计算局部密度差异。

适合：

- 同时考虑距离、时长、点数等多指标的 trip 异常
- 区域画像多指标异常

## Isolation Forest

如果运行环境安装了 `scikit-learn`，脚本可以使用真正的 Isolation Forest。否则会自动跳过并给出 warning，不影响其他算法执行。

# Nice Classification v13 (2026) - Class 34

## 第 34 类 烟草烟具
烟草和烟草代用品；香烟和雪茄；电子香烟和吸烟者用口腔雾化器；烟具；火柴。

【注释】
第三十四类主要包括烟草和烟具，以及某些用于吸烟和烟具的附件和容器。

本类尤其包括：

- 非医用烟草代用品；
- 用于电子烟、吸烟者用口腔雾化器的调味品（香精油除外）；
- 烟用草本植物；
- 鼻烟；
- 某些与烟草和烟具有关的附件和容器，例如：吸烟用打火机，烟灰缸，烟草罐，鼻烟壶，雪茄保润盒。

本类尤其不包括：
- 医用无烟草香烟（第五类）。
- 电子烟用电池和充电器（第九类）；
- 汽车用烟灰缸（第十二类）。

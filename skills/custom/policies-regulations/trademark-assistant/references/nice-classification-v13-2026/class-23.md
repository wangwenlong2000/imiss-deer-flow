# Nice Classification v13 (2026) - Class 23

## 第 23 类 纱线丝
纺织用纱和线。

【注释】
第二十三类主要包括纺织用天然或合成的纱和线。

本类尤其包括：

- 纺织用玻璃纤维线、弹性线、橡胶线和塑料线；
- 绣花、织补和缝纫用线（包括金属线）；
- 绢丝，精纺棉，精纺羊毛。

本类尤其不包括：
- 某些特定用途的线，例如：电线识别线（第九类），外科用线（第十类），贵金属丝线（首饰）（第十四类）；
- 按照材质分类的非纺织品用线，例如：金属捆扎线（第六类）和非金属捆扎线（第二十二类），弹性线、橡胶线或塑料线（第十七类），玻璃纤维线（第二十一类）。

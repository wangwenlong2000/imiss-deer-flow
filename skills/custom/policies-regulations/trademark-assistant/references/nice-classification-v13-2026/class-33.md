# Nice Classification v13 (2026) - Class 33

## 第 33 类 酒
酒精饮料（啤酒除外）；制饮料用酒精制剂。

【注释】
第三十三类主要包括酒精饮料，酒精饮料的原汁和浓缩汁。

本类尤其包括：

- 葡萄酒，加烈葡萄酒；
- 含酒精的苹果酒和梨酒；
- 烈酒，利口酒；
- 酒精饮料原汁，果酒（含酒精），苦味酒。

本类尤其不包括：
- 医用饮料（第五类）；
- 无酒精饮料（第三十二类）；
- 啤酒（第三十二类）；
- 制作酒精饮料用无酒精混合物，例如：软饮料，苏打水（第三十二类）。

# Nice Classification v13 (2026) - Class 27

## 第 27 类 地毯席垫
地毯，地席，亚麻油地毡及其他铺在已建成地板上的材料；非纺织品制壁挂。

【注释】
第二十七类主要包括作为覆盖物铺在已建成的地板和墙壁上的制品。

本类尤其包括：

- 汽车用脚垫；
- 作为地板覆盖物的垫，例如：浴室防滑垫，门前擦鞋垫，体操垫，瑜伽垫；
- 人工草皮；
- 墙纸，包括纺织品制墙纸。

本类尤其不包括：
- 金属制地板、铺地材料和地板砖（第六类），非金属制地板、铺地材料和地板砖（第十九类），木地板条（第十九类）；
- 电热地毯（第十一类）；
- 土工布（第十九类）；
- 婴儿游戏围栏用垫（第二十类）；
- 纺织品制壁挂（第二十四类）。

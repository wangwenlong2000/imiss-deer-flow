# 商标说明撰写指南

商标说明（商标设计说明）是商标注册申请中描述商标特征、构成要素及含义的重要部分。根据《商标法实施条例》第十三条，商标说明需清晰描述商标特征，确保审查员准确理解申请内容。

## 一、图形/组合商标分析流程

对于包含图形元素的商标，必须先使用图像理解工具分析商标图片，再撰写说明。

### 1. 可用的图像理解工具

| 工具 | 类型 | 适用场景 |
|------|------|----------|
| `mcp__zai-mcp-server__analyze_image` | MCP | 通用图像分析，推荐优先使用 |
| `mcp__MiniMax__understand_image` | MCP | 备选方案 |
| `Read` 工具直接读取图片 | 内置 | Claude 原生视觉能力 |

### 2. 分析提示词模板

```text
请详细描述这个商标的构成要素：
1. 商标类型（纯文字/纯图形/组合商标）
2. 文字内容（中文/英文/拼音等）
3. 图形元素（如有）
4. 字体设计特点（普通字体还是艺术字/设计字体，具体设计特征）
5. 颜色使用（如有）
6. 整体设计风格及各元素之间的统一性
```

### 3. 分析要点

从图像分析结果中提取以下关键信息：

- **商标类型**：纯文字 / 纯图形 / 组合商标
- **文字内容**：中文 / 英文 / 拼音 / 数字
- **图形元素**：形状、构图、风格特点
- **字体设计**：普通字体还是艺术字，具体设计特征（弧线、几何化、连笔等）
- **颜色使用**：主色调、配色方案
- **设计风格统一性**：图形与文字的风格是否一致，传达的寓意

### 4. 设计特征描述技巧

#### 字体设计描述

- **弧线元素**：描述哪些字母/笔画使用了弧线设计，弧线传达的视觉效果（灵动、柔和、活力等）
- **几何化处理**：描述字母的几何造型特点（三角形、圆形、菱形等）
- **特殊变形**：描述独特的字母变形（如O内部嵌套图形、字母连笔等）

示例：
> 英文"NOVA ARTS"采用自行设计的艺术字体，全部字母均融入弧线设计元素，与左侧图形的几何造型风格统一。弧线设计寓意生机与活力，传递品牌灵动、创新的特质。

#### 图形设计描述

- **构成元素**：描述图形由哪些基本形状组成
- **构图特点**：对称/不对称、中心/偏心、紧凑/舒展
- **视觉风格**：现代/古典、简约/繁复、科技感/艺术感

示例：
> 左侧为抽象几何图形，由对称的三角形、菱形组合而成，中心嵌有星形图案，整体呈现规整而富有张力的视觉效果。

## 二、文字商标说明的核心内容

### 1. 明确商标类型

直接声明商标的构成要素。

**示例**：

- "本商标由纯中文文字'XX'构成，无图形、字母、数字或其他设计要素。"
- "本商标由中文'XX'及英文'XX'组合构成。"

### 2. 说明文字内容

**普通字体**：
- "商标文字为普通印刷体，无特殊设计。"

**自定义字体/艺术字**：
- "商标文字为自行设计的艺术字体，需保护其独特字形。"

### 3. 解释文字含义

**自创词/无固有含义**：
- "商标文字'XX'为申请人独创，无特定含义。"

**有特殊含义或来源**：
- "商标文字'XX'源自古籍《XXX》，意为'XX'。"

### 4. 非通用名称声明（如涉及）

- "商标文字'XX'非行业通用名称，具有显著性。"

## 三、常见错误避免

- ❌ 无需描述颜色（纯文字商标默认黑白申请，受保护范围更广）
- ❌ 避免使用"著名品牌""驰名商标"等主观表述
- ❌ 不要添加与商标无关的广告语或企业介绍

## 四、实际填写示例

### 案例1：普通字体商标

> 本商标为纯文字商标，由中文"星辰"构成，采用普通黑体字，无图形或其他设计要素。"星辰"为常见词汇，但通过长期使用已具备显著性。

### 案例2：艺术字体商标

> 本商标为纯文字商标，由英文"ZENITH"构成，字体为自行设计的倾斜加粗艺术体，无其他图形或颜色要素。"ZENITH"意为"顶峰"，用于区分商品来源。

### 案例3：自创词商标

> 本商标为纯文字商标，由中文"咔趣"构成，为申请人独创词汇，无固有含义，采用标准宋体字，无图形设计。

### 案例4：组合商标

> 本商标由中文"雅芙"及其拼音"YAFU"构成，两部分均需保护。

### 案例5：放弃部分权利

> 商标中"优选"为行业通用词汇，申请人放弃该部分的专用权。

## 五、特殊情况处理

### 含拼音或外文

需说明是否要求保护拼音/外文部分：
- "商标由中文'雅芙'及其拼音'YAFU'构成，两部分均需保护。"

### 放弃部分权利

若文字中含非显著部分（如"优选""品质"），可声明放弃专用权：
- "商标中'优选'为行业通用词汇，申请人放弃该部分的专用权。"

## 六、官方要求

- 中国商标网申请系统中，"商标说明"字段有字数限制（一般**200字以内**），需简明扼要
- 纯文字商标默认黑白申请，保护范围更广

## 七、生成模板

根据商标类型自动生成说明：

```markdown
## 商标说明

本商标为**{商标类型}**商标，由**{构成要素}**构成，{字体说明}{含义说明}。

**要素构成**：{详细描述各要素}

**含义说明**：{含义或"无特殊含义"}

**其他声明**：{如有放弃权利等情况}
```

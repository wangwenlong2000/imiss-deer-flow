# 公文写作风格与质量指南

## 一、总体风格定位

### 核心原则：行政型而不呆板，流畅易懂又有文采

公文语言应在以下四个维度间取得平衡：
| 维度 | 要求 | 避免 |
|------|------|------|
| **准确** | 概念清晰、表述精确、数据无误 | 模棱两可、含混笼统 |
| **简洁** | 言简意赅、直奔主题、不绕弯子 | 冗长赘述、套话连篇 |
| **庄重** | 用词规范、语气得体、格调端正 | 轻浮随意、网络用语 |
| **流畅** | 逻辑顺畅、过渡自然、可读性强 | 晦涩难懂、故作高深 |

### 语言风格层级
```
公文感 ████████░░ 80%  —— 保持行政公文的正式、规范
可读性 █████████░ 90%  —— 确保读者一目了然
文采度 ██████░░░░ 60%  —— 适度修辞，不过分华丽
理论性 ███████░░░ 70%  —— 体现政策理论水平
```

## 二、行政型语言规范

### 1. 用词选择

#### 推荐用词（行政规范型）
| 场景 | 推荐用词 | 避免用词 |
|------|---------|---------|
| 表示重视 | 高度重视、切实重视 | 很重视、非常重视 |
| 表示推进 | 稳步推进、扎实推进 | 努力做、认真搞 |
| 表示成效 | 取得显著成效、成效明显 | 效果不错、做得很好 |
| 表示落实 | 抓好落实、推动落地 | 把……搞好、弄好 |
| 表示协调 | 统筹协调、联动推进 | 大家一起弄 |
| 表示创新 | 积极探索、勇于创新 | 试试看、搞点新花样 |
| 表示问题 | 仍然存在、亟待解决 | 问题还不少、有待改进 |
| 表示决心 | 坚定不移、毫不动摇 | 一定要、肯定能 |

#### 活力词汇（增加可读性）
- "打通××的'最后一公里'"
- "激活××的内生动力"
- "打造××的升级版"
- "形成××的强大合力"
- "让××落地生根、开花结果"
- "按下××的'快进键'"
- "跑出××的'加速度'"

#### 适度修辞（提升文采）
- 排比句（三句式最佳）：
  - "注重……、突出……、强化……"
  - "以……为引领、以……为支撑、以……为保障"
- 对偶式表达：
  - "既要'面子'更要'里子'"
  - "不搞'一刀切'、不求'齐步走'"
- 比喻（恰当使用）：
  - "补短板、强弱项、堵漏洞"
  - "筑牢……防线"

### 2. 句式运用

#### 长短句搭配
- **短句**：陈述结论、强调重点
  - "成效显著。""问题突出。""任务艰巨。"
- **长句**：说明原因、阐述政策
  - "按照……要求，结合……实际，围绕……目标，着力推进……建设。"

#### 主动句为主
- 推荐："我局积极推进××改革"
- 避免："××改革被积极推进"

#### 肯定句为主
- 推荐："各项工作取得显著成效"
- 避免："各项工作未出现明显问题"（双重否定，阅读理解成本高）

### 3. 段落组织

#### 段落长度控制
- 每段150-250字为宜
- 最长不超过300字
- 短段落（50-100字）用于强调和过渡

#### 段落逻辑结构
```
【总—分—总】模式（最常用）
├── 总：段首句概括段落要点（主题句）
├── 分：展开说明、数据支撑、具体做法
└── 总：段尾句小结或过渡到下一段
```

#### 段落间的过渡
- 顺承："在此基础上……""与此同时……"
- 转折："尽管如此……""同时也要看到……"
- 递进："更为重要的是……""特别是在……方面"
- 总结："总的来看……""综合以上情况……"

## 三、逻辑结构设计

### 1. 请示的逻辑结构
```
为什么要请示（缘由）
    ↓  政策依据 → 现实需求 → 紧迫性
要请示什么（事项）
    ↓  方案要点 → 实施计划 → 资源需求
请示的依据是什么（支撑）
    ↓  政策支撑 → 可行性分析 → 预期效果
```

### 2. 报告的逻辑结构
```
做了什么（做法）
    ↓  组织部署 → 具体措施 → 创新做法
效果如何（成效）
    ↓  数据支撑 → 典型案例 → 横向对比
还存在什么问题（不足）
    ↓  客观分析 → 深层原因 → 影响判断
下一步怎么做（打算）
    ↓  目标导向 → 措施安排 → 时间节点
```

### 3. 论证逻辑方法

#### 政策论证
```
上级有要求 → 本地有需要 → 实施有基础 → 推进有保障
（必要性）    （迫切性）    （可行性）    （可靠性）
```

#### 成效论证
```
定性描述（取得显著成效）
    + 定量数据（完成率××%，增长××%）
    + 横向对比（位列全省/全市第×位）
    + 群众反响（满意度××%）
```

#### 问题分析
```
现象描述 → 原因分析 → 影响评估 → 应对思路
（是什么）   （为什么）   （有何影响）  （怎么办）
```

## 四、标题撰写深度指南

### 1. 标题对仗的核心技法

#### 技法一：结构对称
```
动宾 + 动宾
深化 ×××改革  |  推进 ×××建设
强化 ×××管理  |  提升 ×××水平

状动宾 + 状动宾
全面推进 ×× |  持续深化 ××
着力破解 ×× |  全力推进 ××
```

#### 技法二：字数均等
```
四字对：深化改革 / 完善机制
六字对：强化组织领导 / 健全工作机制
八字对：聚焦重点难点 / 突出关键环节
```

#### 技法三：语义互补
```
并列关系：统筹城乡发展  协调区域建设（横向）
递进关系：深化体制改革  加快制度创新（纵向）
因果关系：适应新形势    推动新发展（因果）
目的关系：聚焦高质量    服务现代化（手段→目的）
```

### 2. 标题常见问题修正

| 问题标题 | 修正建议 |
|---------|---------|
| 关于工作的报告 | 太笼统，应明确具体工作内容 |
| 关于加快推进XX工作的请示 关于进一步加强XX管理的报告 | "加快推进""进一步加强"过于常见，缺乏特色 |
| 关于XX局XX科XX同志职务任免的请示 | 太具体琐碎，作为附件更合适 |
| 关于认真学习贯彻XX精神 深入开展XX活动 切实加强XX建设的报告 | 三句式过长，影响可读性，建议精简 |

## 六、特殊注意事项

### 1. 敏感词规避
- 涉及民族、宗教、台港澳等敏感话题时，严格使用标准表述
- 引用领导人讲话时，逐字核对准确性
- 使用地名、机构名称时，使用全称或规范简称

### 2. 称谓规范
- 上级机关：使用全称
- 并列单位：按级别排列
- 人员称谓：姓名+职务（首次出现时）

### 3. 语气把握
- 向上级汇报：谦和、客观、求实
- 请示事项：诚恳、明确、有理有据
- 反映问题：实事求是、不夸大不缩小
- 提出建议：积极、具体、可行
